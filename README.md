## Getting Started

### Prerequisites

- Windows 7 or later
- Administrator privileges (for full functionality)

### Installation

1. Download the toolkit files
2. Extract to your preferred directory
3. Run either:
   - `gon_cyber_toolkit_en.bat` (English)
   - `gon_cyber_toolkit_pt.bat` (Portuguese)
